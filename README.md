# AI Code Mentor  
### Explain & Optimize Code Instantly with AI

AI Code Mentor intelligently analyzes any code you provide, breaks down its logic, highlights potential inefficiencies, and generates a cleaner, faster, and more maintainable optimized version—ensuring developers understand their code better while improving its overall quality.

---

## Features

- **AI Code Explanation** – Understand any code in simple terms  
- **AI Code Optimization** – Returns a faster, cleaner version of your code  
- **Language Auto-Detection** – Works for JS, Python, C++, Java, HTML/CSS & more  
- **CodeMirror Editor** – Smooth, modern coding experience  
- **Responsive UI** – Sidebar for modes (Explain/Optimize)  
- **Upload & History** (upcoming)  

---

## Tech Stack

- **Next.js 16 (App Router)**
- **React**
- **Tailwind CSS**
- **Groq SDK (Llama Models)**
- **CodeMirror**
