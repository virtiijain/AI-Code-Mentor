# AI Code Mentor  

AI Code Mentor intelligently analyzes any code you provide, breaks down its logic, highlights potential inefficiencies, and generates a cleaner, faster, and more maintainable optimized version. Now, with AI-powered text-to-speech, you can even listen to AI-generated explanations for a hands-free learning experience.
[Try it here](https://ai-powered-code-analyzer.vercel.app/)

## Features

- **AI Code Explanation** – Understand any code in simple terms  
- **AI Code Optimization** – Returns a faster, cleaner version of your code
- **AI-Powered Text-to-Speech** – Listen to explanations instead of reading
- **AI Bug Detector** – Automatically highlights potential bugs in your code
- **Flowchart & Diagram Generation** – Visualize your code logic with flowcharts 
- **Language Auto-Detection** – Works for JS, Python, C++, Java, HTML/CSS & more  
- **CodeMirror Editor** – Smooth, modern coding experience  
- **Responsive UI** – Sidebar for modes
